# Legal Intelligence System

A full-stack MERN application that allows users to upload legal documents and automatically analyze them for insights such as credibility score, summaries, and keywords.

---

## 🚀 Features

- User Authentication (JWT based login/register)
- Secure backend APIs
- Document upload system
- AI-based legal document analysis
- Credibility scoring system
- Document summary generation
- Keyword extraction
- Personal dashboard for each user
- Multi-user support

---

## 🛠 Tech Stack

### Frontend
- React
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Tokens)
- bcrypt password hashing

### Other Tools
- Multer (file upload)
- Nodemon

## ⚙️ How to Run Locally

### 1️⃣ Clone Repository


git clone https://github.com/aaryan-purohit-2465/Legal-Intelligence-System


### 2️⃣ Install Backend


cd backend
npm install
npm run dev


### 3️⃣ Install Frontend


cd frontend
npm install
npm run dev


### 4️⃣ Start MongoDB

Make sure MongoDB is running locally.
